Why Choose Us — Section Header

Feature Cards

Each card shows an icon, title, and description in the "Why Choose Us" grid. Icon names use Font Awesome classes (e.g. fas fa-bed).

Rooms Section Header

Room Cards

Each room card appears in the Rooms section. You can add, edit, or delete rooms. The "Tag Style" field accepts: popular, luxury, or leave empty for default dark style.

Pricing Section Header

Pricing Plans

Each pricing card has a type, 3 prices (Weekday/Weekend/Holiday), and a list of features. Toggle "Featured" to highlight a card. Enabled features show a checkmark, disabled features show an ✗.

Offers Section Header

Offer Cards

Badge Style accepts: (empty for gold), "weekend" for dark, or "family" for green. End Date controls the countdown timer shown on the card.

Services Section Header

Service Cards

Toggle the checkbox to show or hide each service on the website. Use Font Awesome icon class names (e.g. "fas fa-wifi") in the Icon field.

Gallery Section Header

Gallery Images

The first image will display larger (spanning 2 columns). Add a caption for each photo — it appears in the lightbox when the image is clicked.

Reviews Section Header

Guest Reviews

Click the stars to set the rating for each reviewer. Reviewer photos can be URLs to any image (e.g. from randomuser.me or your own photos).

FAQ Section

Each FAQ item has a question and an answer. Add, edit, or remove items freely.
